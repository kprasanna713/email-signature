# Email Signature - Prasanna Kumar V

Professional HTML email signature with clickable social icons.

## Setup

1. Add your profile photo as `images/profile.jpg`
2. Push to GitHub and enable GitHub Pages (main branch)
3. Visit `https://kprasanna713.github.io/email-signature/`
4. Click "Copy Signature" and paste into your email client settings
